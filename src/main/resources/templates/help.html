# URL Méthode attendue Description
# / GET La page d’accueil
# /help GET Affiche la page contenant les informations d’aide
# /rss22/resume/xml GET Affiche la liste des articles stockés
# /rss22/resume/html GET Page HTML (ou XHTML)
# /rss22/resume/xml/{guid} GET Affiche le contenu complet de l’article dont l’identifiant est {guid}
# /rss22/html/{guid} GET Affiche le contenu complet de l’article dont l’identifiant est {guid}
# rss22/insert POST Le flux reçu est validé par le schéma XSD de définition rss22
# /rss22/delete/{guid} DELETE Suppression de l’article dont l’identifiant est {guid}
